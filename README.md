# Calculadora de IMC

Uma calculadora simples de IMC (Índice de Massa Corporal) desenvolvida em **HTML, CSS e JavaScript**.  
Permite ao usuário inserir nome, idade, peso e altura para calcular o IMC e receber uma classificação de acordo com os valores.

## 🚀 Funcionalidades
- Entrada de **nome, idade, peso e altura**
- Navegação entre campos usando a tecla **Enter**
- Cálculo do IMC com classificação:
  - Abaixo do peso
  - Peso normal
  - Sobrepeso
  - Obesidade
- Interface amigável com fundo ilustrativo e cores diferentes para cada resultado
